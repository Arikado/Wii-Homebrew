Default Keyboard Controls

Keyboard | Joystick | Level-Editor

Keyboard

Moving Around

Moving Left
= LEFT
Moving Right
= RIGHT

Actions

Jump
= UP
Duck
= DOWN
Shoot
= SPACE
Get Item from Itembox
= ENTER

Misc

Open Main menu
= ESC
Quit ( in the Main menu )
= ESC
Make a Screenshot
= F5
Enable/Disable Sounds
= F10
Enable/Disable Music
= F11
Load a New Level
= L

Joystick

Moving Around

Moving Left
= JOY LEFT
Moving Right
= JOY RIGHT

Actions

Jump
= JOY BUTTON 0
Duck
= JOY DOWN
Shoot
= JOY BUTTON 1
Get Item from Itembox
= JOY BUTTON 3

Level Editor

Normal Operations

Open/Close Leveleditor
= F8
Save Current changes
= CTRL + S
Copy the selected Object
= CTRL + C
Paste the copied Object
= CTRL + V
Modify Object to passive/massive/halfpassive
= M
Camera to the level start
= HOME
Camera centers the last Levelexit
= END

Precision Positioning

Shift/Move Left
= NUMPAD 4
Shift/Move Right
= NUMPAD 6
Shift/Move Up
= NUMPAD 8
Shift/Move Down
= NUMPAD 2

Fast Map Scrolling

Jump Right One Screen
= N
Jump Left One Screen
= P

Mouse Control

Move Object
= LEFT-CLICK AND DRAG MOUSE
*Fastcopy Mode*
= MIDDLE-CLICK AND NUMPAD DIRECTIONS (like Precision Positioning)
Camera Mode
= MIDDLE-BUTTON (with nothing selected)
Delete Object
= RIGHT-CLICK
Continuous Delete - Everything that Touches the Mouse
= CTRL + RIGHT-CLICK
